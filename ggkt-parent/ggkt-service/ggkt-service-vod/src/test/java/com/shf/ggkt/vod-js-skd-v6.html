UGC-Uploader

1.示例中的签名是直接从demo后台获取签名。该 demo 对应的点播账号已开启防盗链功能，此视频分发 URL 仅限 2 个独立 IP 进行访问，有效期为 10 分钟。
2.示例1点击“直接上传视频”按钮即可上传视频。
3.示例2点击“添加视频”添加视频文件，点击“添加封面”添加封面文件，然后点击“上传视频和封面”按钮即可上传视频和封面。
4.取消上传为取消上传中的视频，上传成功的视频不能取消上传。
